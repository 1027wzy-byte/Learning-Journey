# 一、Java 新版日期时间 API（java.time）

> JDK8 开始推荐使用 `java.time` 包，替代旧版 `Date`。

## 1. 三个核心类

| 类 | 作用 |
|------|------|
| LocalDate | 日期（年/月/日） |
| LocalTime | 时间（时/分/秒） |
| LocalDateTime | 日期+时间 |

### 创建对象

```java
LocalDate date = LocalDate.now();
LocalTime time = LocalTime.now();
LocalDateTime dateTime = LocalDateTime.now();

LocalDate birthday = LocalDate.of(2028,7,12);
```

输出：

```
2028-07-12
```

---

## 2. 日期计算

日期对象不可修改，每次计算都会返回新的对象。

```java
LocalDate date = LocalDate.of(2028,7,12);

LocalDate newDate = date.plusDays(5);

System.out.println(date);
System.out.println(newDate);
```

结果：

```
2028-07-12
2028-07-17
```

常用方法：

```java
plusDays()

plusMonths()

plusYears()

minusDays()

minusMonths()

minusYears()
```

---

## 3. Duration（计算时间差）

适用于：

- 程序运行时间
- 接口耗时
- 秒、毫秒计算

```java
Instant start = Instant.now();

// 执行代码

Instant end = Instant.now();

Duration duration = Duration.between(start,end);

System.out.println(duration.toMillis());
```

常用：

```
toMillis()

toSeconds()

toMinutes()

toHours()
```

---

## 4. Period（计算日期差）

适用于：

- 年龄
- 两个日期之间差多少年/月/日

```java
LocalDate birth = LocalDate.of(2000,5,20);

LocalDate today = LocalDate.now();

Period p = Period.between(birth,today);

System.out.println(p.getYears());
System.out.println(p.getMonths());
System.out.println(p.getDays());
```

# 二、DateTimeFormatter（日期格式化）

## 1. 创建格式器

```java
DateTimeFormatter formatter =
    DateTimeFormatter.ofPattern("yyyy/MM/dd HH:mm:ss");
```

---

## 2. 日期对象 → 字符串

```java
LocalDateTime now = LocalDateTime.now();

String str = now.format(formatter);
```

结果：

```
2026/07/15 17:30:00
```

---

## 3. 字符串 → 日期对象

```java
String str = "2026/07/19 17:18:12";

LocalDateTime time =
        LocalDateTime.parse(str,formatter);
```

---

## 常见格式

```
yyyy-MM-dd

yyyy/MM/dd

yyyy-MM-dd HH:mm:ss

yyyy/MM/dd HH:mm:ss
```

# 三、为什么推荐 java.time？

旧版 Date：

❌ 年份需要减1900

❌ 月份从0开始

❌ 线程不安全

❌ API设计混乱

新版：

✅ API清晰

✅ 不可变对象

✅ 线程安全

✅ 功能完整

因此：

> 企业开发基本都使用 **java.time**。

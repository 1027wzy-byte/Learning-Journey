# 📚 今日学习内容

今天继续学习 Java 面向对象相关知识，重点掌握了 JavaBean、包管理、访问控制修饰符，以及 static、final 的使用。

## 1. JavaBean

学习内容：

- JavaBean 规范
- private 成员变量
- Getter / Setter
- 无参构造方法

掌握内容：

- 学会按照标准规范封装对象。
- 理解为什么成员变量需要私有化。
- 明白 Getter / Setter 是访问对象数据的统一入口。

---

## 2. package 与 import

学习内容：

- package 声明包
- import 导入类

掌握内容：

- 理解包的作用是避免类名冲突。
- 学会导入不同包中的类。

---

## 3. 访问控制修饰符

学习内容：

- public
- protected
- 默认(default)
- private

掌握内容：

- 理解四种访问权限的区别。
- 明白封装的核心思想：隐藏数据，对外提供访问接口。

---

## 4. static 与 final

学习内容：

- 静态变量
- 静态方法
- 静态代码块
- static final 常量
- final 修饰变量、方法、类

掌握内容：

- 区分实例成员与静态成员。
- 理解 static 属于类，所有对象共享。
- 理解 final 用于限制修改，提高程序安全性。

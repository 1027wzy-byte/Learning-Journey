## 📚 课堂重点

### Agent

Agent 是能够根据目标自主规划、执行任务的 AI。

它与普通聊天 AI 最大的区别是：

- 能自主完成任务
- 能连续执行多个步骤
- 能自动调整执行过程

---

### Java 开发环境

JDK = JRE + 开发工具

JRE = JVM + Java 类库

JVM 负责运行 Java 程序。

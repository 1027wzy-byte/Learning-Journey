# 📚 今日学习内容

今天继续学习 Java 面向对象，主要围绕继承、super、方法重写展开。

## 1. super 关键字

学习内容：

- super 访问父类成员变量
- super 调用父类方法
- super() 调用父类构造方法

掌握内容：

- 创建子类对象时会先执行父类构造方法。
- super() 必须写在构造方法第一行。
- 当父子类成员同名时，可以使用 super 区分父类成员。

---

## 2. 方法重写（Override）

学习内容：

- 方法重写规则
- 重写后的调用机制

掌握内容：

- 重写要求方法名、参数列表完全一致。
- 返回值类型保持兼容。
- 调用重写方法时遵循 **new 谁调用谁** 的原则。

---

## 3. 抽象类（Abstract）

学习内容：

- abstract 方法
- abstract 类

掌握内容：

- 抽象方法没有方法体。
- 抽象类不能直接创建对象。
- 子类必须实现所有抽象方法。

---

## 4. 接口（Interface）

学习内容：

- interface
- implements
- 接口实现

掌握内容：

- 接口中主要包含抽象方法。
- 一个类可以实现多个接口。
- 接口方法实现时必须使用 public。

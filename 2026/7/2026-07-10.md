# 📚 今日学习内容

今天正式进入 Java 面向对象进阶内容，主要学习了继承、多态以及抽象类等核心知识。

## 1. super 与方法重写

学习内容：

- super 关键字
- 调用父类成员变量
- 调用父类方法
- 调用父类构造方法
- 方法重写（Override）

掌握内容：

- super() 必须位于构造方法第一行。
- Java 创建子类对象时会先创建父类对象。
- 重写要求方法名、参数列表完全一致。
- 重写方法调用遵循 **new 谁调用谁** 的原则。

---

## 2. 抽象类与接口

学习内容：

- abstract 方法
- abstract 类
- interface 接口
- implements
- 接口实现

掌握内容：

- 抽象类不能实例化。
- 抽象方法没有方法体。
- 子类必须实现所有抽象方法。
- 一个类可以实现多个接口。
- 接口中的抽象方法必须使用 public 实现。

---

## 3. 多态

学习内容：

- 向上转型
- 向下转型
- instanceof
- 多态应用

掌握内容：

- 父类引用可以指向子类对象。
- 能访问什么成员由引用类型决定。
- 重写方法执行由对象实际类型决定。
- 向下转型前建议使用 instanceof 判断。
- 多态能够提高代码复用性。

---

## 4. 匿名内部类

学习内容：

- 匿名内部类
- 接口匿名实现
- 抽象类匿名继承

掌握内容：

- 对象只使用一次时可以采用匿名内部类。
- 不需要额外创建新的 Java 类。
- 能够快速创建接口实现类或抽象类子类对象。
